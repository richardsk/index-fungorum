Index Fungorum Registration Help
Select the type of nomenclatural act you are registering from the list - the one selected is not retained (the list box returns to a blank) but the title of the form below the list box changes to show the nomenclatural act selected and displays the appropriate field set: below are notes to help you complete the form so it passes the validation
Form box Help notes
Name Enter the name, including any infraspecific or infrageneric rank term, but EXCLUDING any author citations
Rank Select a rank if the form deals with more than one rank
Author Enter the full author citation, including basionym authors, just as you would in manuscript preparation
Publishing Authors Enter the author(s) of the publication in which the name will be published ONLY if this differs from the name authors entered above
Journal or Book Either select from the list (the journals listed are those which publish the largest number of names) or enter the details in the box below
Volume The journal or book volume number
Part The part number if the volume is split; IF policy is to add 'no.' if the pagination is non sequential (i.e. each part starts with page 1)
Page The single page on which the name being registered will be published; this field is mandatory so if unknown use '1'
Year of publication This is the year that the name will be effectively published; if unknown use the current year
Year on publication This is the year printed on the journal or book ONLY if it differs from the one above
Host/Associated organism/ecosystem This is not mandatory but useful information for taxonomists; keep the information concise
Location This is not mandatory but useful information for taxonomists; select from the list of TDWG Geographical Recording System areas or an ISO country
Basionym Look up the name of the basionym in the pop-up window but only enter the IF number
Replaced synonym Look up the name of the replaced synonyms (the illegitimate name, epithet bringing name or unavailable name) in the pop-up window but only enter the IF number
Competing synonym Look up the name of the competing synonym in the pop-up window but only enter the IF number
Typification details Include only the Code required data - the Index Herbariorum acronym or Institution or Reference Collection name where the type will be preserved + the accession number or the collector and the collection number/date. For new genera register the genus first then the type species
Diagnosis Include either a diagnosis (preferred) OR the full description, in Latin or English
Notes Add any notes (these are not visible) which might be useful to explain things, for example, etymology
Email Add your email address; this will be used to send you the identifier which must be cited in the protologue associated with the name; this is also used to identify the names you registered if you are intending to use the Index Fungorum instant publication system. If you do not use the same email address for your username login your names will not be accessible
