Index Fungorum - register and publish name

     

Le système IF d'inscription est séparée de la IF système de publication; lorsque vous enregistre les noms de (noms de nouveaux taxons, de nouvelles combinaisons, de nouveaux noms) ou typifications l'adresse e-mail que vous utilisez pour obtenir la identificateur identifie également ces noms et typifications que votre proche. Lorsque vous créez un compte de publier avec la même adresse de courriel de cette donne accès à «votre» nom de publier. Je vous souhaite d'ajouter un nom manquant Utilisez la page «ajouter le nom manquant».

Pour enregistrer un nom qui sera bientôt publié, cliquez ici; Pour ajouter un nom qui est déjà publié, mais absent de IF, cliquez ici. S'il vous plaît ne pas enregistrer des noms de qui ont été précédemment enregistrés avec d'autres banques – par exemple MycoBank – car cela crée un doublon, et après le 1er Janvier 2013 pourrait se traduire par un nom invalide.

Après vous être inscrit noms et typifications vous sera envoyé par courriel un identifiant Fungorum indice, qui doit être cité dans le protologue. Noms seront examinés avant qu'ils ne deviennent visibles dans les résultats de recherche. Si aucun détails de la publication sont à condition que le nom ne sera pas publié. Si le manuscrit est «sous presse» ou actuellement publié en ligne qu'à suivre les instructions pertinentes disponibles à partir de la page d'inscription. Lors de l'enregistrement des noms ou typifications les détails bibliographiques nécessaires sont ceux où vous avez l'intention de publier les noms et typifications vous enregistrant, PAS la référence basionyme, ou la référence synonyme remplacé ou, pour la typification, la référence à la dénomination que vous typifiant – tous ces les détails sont déjà disponibles dans l'index Fungorum et seront récupérés lors de l'examen pré-publication. Si elles ne sont pas de présenter les documents pertinents doivent être mis à jour avant la publication.

Vous pouvez publier vos noms immédiatement après l'enregistrement d'un nom ou les noms, mais cela nécessite un login – si vous n'avez pas de login de l'utilisateur en créer un ici, vous pouvez vous connecter ici. Vous pouvez consulter toutes les publications précédente IF peut être consulté et téléchargé ici.

Allocations d'identificateurs dans Index Fungorum, MycoBank et d'autres dépôts peuvent être consultés ici. Les identifiants sont uniques pour toute indépendante nom de l'endroit où ils ont été émis. Par exemple, un identifiant délivré par Index Fungorum permettra de résoudre via une URL Index Fungorum –www.indexfungorum.org/Names/NamesRecord.asp?RecordID=550000 – et une adresse URL MycoBank –http://www.mycobank.org/MycoTaxo.aspx?Link=T&Rec=550000.
