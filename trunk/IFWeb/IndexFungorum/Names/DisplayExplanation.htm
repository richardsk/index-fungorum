field help notes
Name If you are adding a name of a species (or below) use IndexFungorum to check the genus name if necessary
Author Check the authors of fungal names database so that the correct form of the authors names is used
Title of Journal or Book If it's not in the list add details to Notes field; IndexFungorum uses a simple relational model to store journal and book titles
Year of publication This is the year of effective publication - see Art. 31
Year on publication This is the year as printed on the publication where it differs from the year of effective publication
Synonym Either add the name of the basionym (for combinations) or the replaced synonym (for new names) OR use the search function to find the IndexFungorum number and add this
Country of Origin of type If it's not in the list add details to Notes field
Type details Add code compliant details - see Art. 37.3 Note 2
